Howto edit the site

1. Become a contributor

  • Create a github account: http://github.com.
  • Ask for permission to edit the website: mail me at patrick dot hochstenbach at ugent dot be

2. Choose one of these

A. For the less techies

Go to https://github.com/elag/elag.github.io and change the HTML code manually

B. For the real techies

Checkout the github project: 

$ git clone https://github.com/elag/elag.github.io

Change to the 'elag.github.io' directory: 

$ cd elag.github.io

Create, edit, delete the files. And dont forget the basics of git: 

# When you add a new file or directory to this project
$ git add file
$ git add directory

# When you delete a file from this project
$ git rm file

And of course, tell us of all the changes you have made: 

# Commit the changes
$ git commit -a
# Check if you are still up to date with the remote repository
$ git pull
# Push the changes to the remote repository
$ git push

That's it :-)!
